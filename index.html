ประวัติส่วนตัว

สวัสดีค่ะ ดิฉันเป็นนักเรียนที่มีความสนใจด้านวิศวกรรมศาสตร์ มีความมุ่งมั่นในการเรียนรู้ พัฒนาทักษะด้านวิทยาศาสตร์ คณิตศาสตร์ เทคโนโลยี และการเขียนโปรแกรม

ชื่นชอบการสร้างเว็บไซต์ ออกแบบ UI เขียน HTML CSS JavaScript และสนใจด้าน Automation, AI รวมถึง Internet of Things (IoT)

มีความรับผิดชอบ ทำงานเป็นทีมได้ดี สามารถเรียนรู้สิ่งใหม่ได้รวดเร็ว พร้อมพัฒนาตนเองอยู่เสมอ

งานอดิเรก ได้แก่ อ่านหนังสือ ดูคลิปความรู้ ออกแบบเว็บไซต์ สร้างโปรเจกต์ใหม่ ๆ และฝึกเขียนโปรแกรมทุกวัน
