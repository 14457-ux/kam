---

## 📸 เปลี่ยนรูป

นำรูปของคุณไปแทนไฟล์

```
assets/profile.jpg
```

สามารถเปลี่ยนชื่อไฟล์ได้

แต่ต้องแก้ใน

```
index.html
```

เช่น

```html
<img src="assets/profile.jpg">
```

---

## 🏆 เพิ่มเกียรติบัตร

นำรูปไปใส่ใน

```
assets/
```

แล้วเพิ่ม

```html
<img src="assets/certificate3.jpg">
```

---

## 💼 เพิ่มผลงาน

คัดลอก

```html
<div>
<img src="assets/project1.jpg">
<h3>Project</h3>
</div>
```

แล้วเปลี่ยนรูปและข้อความได้เลย

---

## 📱 เปลี่ยนช่องทางติดต่อ

แก้ไขใน

```html
<section id="contact">
```

เช่น

```html
example@gmail.com
```

เปลี่ยนเป็นอีเมลของคุณ
